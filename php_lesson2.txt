Объедините их в один массив и отсортируйте по возрастанию
<?php

4.Фильтрация чисел в массиве:
Задание:Создайте массив с числами. Оставьте только те элементы, которые больше 10.
<?php
$arr1 = array(1,0,6,12,55);

$arrNew = array_filter($arr1, function($value){
	if ($value>10) return $value;
});
print_r($arrNew);
?>

5.Обход массива с использованием цикла for:
Задание:Создайте массив чисел. Используйте цикл for для вывода элементов массива на экран.
<?php
$arr1 = array( 10 , 22 , 45 , 36 );
$counter = count($arr1);
for($i = 0; $i < $counter; $i++) {
	echo "$arr1[$i] \r";
}

?>

7.Функция для определения длины строки:Напишите функцию,которая принимает строку в качестве аргумента и возвращает её длину (количество символов).Вызовите эту функцию с разными строками и выведите результаты.strlen($string)
<?php
function evenNumber($string){
	return strlen($string)-2;

};

$arr1 = array('Toma','','Tamara');
$counter = count($arr1);
for($i = 0; $i < $counter; $i++) {
	$value= "$arr1[$i] \r";
	$result = evenNumber ($value) ;
echo "$result \r";
};

?>
8.Функция для проверки четности числа: Напишите функцию, которая принимает число в качестве аргумента и проверяет, является ли оно четным. Функция должна возвращать true, если число четное, и false, еслине четное. Вызовите эту функцию с разными числами и выведите результаты.
<?php
function evenNumber($number){
	if ($number % 2 === 0 )
	return 'true';
	else return 'false';
};


$arr1 = array( 10 , 22 , 45 , 36 );
$counter = count($arr1);
for($i = 0; $i < $counter; $i++) {
	$value= "$arr1[$i] \r";
	$result = evenNumber ($value) ;
echo "$result \r";
};

?>
